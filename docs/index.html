CreateQueueRequest

CreateQueueRequest represents a request to create a queue.

Field Type Label Description
queue_name string

queue_name represents the name of the queue.

retention_period_seconds uint64

retention_period_seconds specifies the time after which messages in the queue expire.

visibility_timeout_seconds uint64

visibility_timeout_seconds is the time a message stays invisible after being read.

max_receive_attempts uint32

max_receive_attempts defines the maximum number of times a message can be read before it is deleted.

eviction_policy EvictionPolicy

eviction_policy defines queue behaviour when the retention_period_seconds or max_receive_attempts is reached.

dead_letter_queue_id string

dead_letter_queue_id is taking effect only when the policy is set to DeadLetter.

EvictionPolicy

EvictionPolicy represents an enumeration of supported policies.

Name Number Description
EVICTION_POLICY_DROP 0

EVICTION_POLICY_DROP is a default policy. When queue has to apply the eviction policy to the message, the message will be dropped (permanently removed) from the queue.
EVICTION_POLICY_DEAD_LETTER 1

EVICTION_POLICY_DEAD_LETTER represents a policy that when applied to the message, the message will be moved to another queue, which has been specified as dead letter queue.
EVICTION_POLICY_REORDER 2

EVICTION_POLICY_REORDER represents a policy that when applied to the message, the message will be reinserted in the queue.
